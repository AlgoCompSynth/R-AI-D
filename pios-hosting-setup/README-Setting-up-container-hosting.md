# Setting up Container Hosting

This directory contains scripts for setting up container hosting. I currently
test only on Raspberry Pi OS 64-bit, which is based on Debian 13 "trixie", and
on Bluefix DX, which has container hosting built in. To set up container
hosting on a Raspberry Pi, run the two scripts

1. `./1_apt_packages.sh`
2. `./2_command_line_setup.sh`

in that order.
